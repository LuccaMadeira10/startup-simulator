# Startup Simulator

##  Sobre o Projeto

O **Startup Simulator** é uma aplicação web full stack que simula a criação e gestão de uma startup ao longo do tempo, com diversas funcionalidades.

A ideia do projeto é permitir que um usuário crie uma startup a seu critério e tome decisões estratégicas a cada ciclo de simulação (por exemplo, mensalmente, anual etc). Essas decisões podem envolver áreas como marketing, desenvolvimento de produto e contratação de equipe, e financeira.

Com base nessas decisões, o sistema calcula automaticamente diferentes métricas de negócio, como crescimento de usuários, receita, custos e capital disponível.

O objetivo é observar como diferentes estratégias podem impactar o crescimento de uma startup ao longo do tempo.

##  Ideia Geral da Aplicação

A aplicação permitirá que o usuário:

- Crie uma startup
- Tome decisões estratégicas a cada rodada
- Execute ciclos de simulação
- Visualize métricas de crescimento
- Analise o histórico de decisões e resultados

O sistema calculará automaticamente indicadores como:

- Crescimento de usuários
- Receita
- Custos operacionais
- Capital disponível
- Evolução da empresa ao longo do tempo
